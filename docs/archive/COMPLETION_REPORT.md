## 🚀 如何使用

### 启动应用
```bash
cd /path/to/ai-novel-generator-4.0
python -m src.ui.app
```

### 功能使用流程
1. **配置API**: ⚙️ 系统设置 > 🌐 接口管理
2. **创建项目**: 📖 小说创作 > 新建项目
3. **生成大纲**: 📖 小说创作 > ✍️ 大纲生成
4. **生成章节**: 📖 小说创作 > 生成章节
5. **润色内容**: ✨ 小说润色 > 选择文本和润色类型
6. **重写内容**: 📝 小说重写 > 选择文本和风格
7. **导出作品**: 💾 导出与分享 > 选择格式导出
8. **管理缓存**: ⚙️ 系统设置 > 💾 缓存管理
