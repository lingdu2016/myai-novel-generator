## 快速开始

### 第一次使用

#### 1. 启动应用

```bash
python run.py
```

应用将在 http://127.0.0.1:7860 启动

#### 2. 配置API提供商

**推荐方案：使用Ollama（本地免费）**

1. 打开"API配置"标签
2. 选择"🦙 Ollama (本地)"
3. 点击"🔗 测试连接"
4. 点击"💾 保存配置"

**或使用云端服务**

1. 打开"API配置"标签
2. 选择提供商（如"🤖 OpenAI"）
3. 输入API Key
4. 点击"🔗 测试连接"
5. 点击"💾 保存配置"

#### 3. 创建第一个项目

在"小说创作"标签中：

```
小说标题: 我的修仙之旅
类型: 玄幻仙侠
人物设定: 
  主角：张三，性格坚毅，出身贫寒
世界观: 
  修仙世界，灵气充沛
  境界：练气、筑基、金丹、元婴...
主线剧情: 
  少年张三意外获得神秘玉佩，
  开始踏上修仙之路
计划章节数: 50
```

点击"✨ 创建项目"

## 项目创建

### 基本信息填写

#### 小说标题
- 简洁明了的标题
- 示例："都市神医"、"星际霸主"、"我的修仙之旅"

#### 类型
- 选择小说类型
- 常见类型：玄幻仙侠、都市言情、科幻硬核、悬疑惊悚等
- 可自由填写

#### 人物设定
- 描述主要角色
- 包括：性格、背景、能力等
- 示例：
  ```
  主角：张三，25岁，性格坚毅，出身贫寒
  配角：李四，张三的好友，性格开朗
  反派：王五，阴谋家，心机深沉
  ```

#### 世界观
- 描述故事背景和规则
- 玄幻类：修仙体系、魔法系统、种族设定
  ```
  修仙体系：练气→筑基→金丹→元婴→化神
  世界等级：下界、中界、上界
  灵气浓度：影响修炼速度
  ```
- 都市类：现实世界背景
  ```
  时间：2025年
  地点：虚构的大都市
  社会背景：现代都市
  ```

#### 主线剧情
- 故事的主要走向
- 示例：
  ```
  起因：主角获得神秘传承
  发展：拜师学艺、历练冒险
  高潮：对抗强大敌人
  结局：成为巅峰强者
  ```

#### 计划章节数
- 整个小说预计的章节数
- 建议：短篇10-30章，中篇50-100章，长篇100+章

## 章节生成

### 生成单个章节

#### 步骤1：填写章节信息

```
章节号: 1
章节标题: 初入仙门
章节大纲: 
  张三在山中采药时，意外发现一古老洞府。
  在洞府中获得了修仙传承，决定前往青云宗拜师。
  路上遇到劫匪，展现出初学的仙法。
目标字数: 3000
☑ 启用连贯性系统
```

#### 步骤2：生成

点击"🚀 生成章节"按钮

#### 步骤3：查看结果

- **生成内容**：显示AI生成的章节内容
- **状态信息**：显示生成状态、字数等
- **连贯性验证**：显示连贯性评分和详细报告

### 批量生成（规划中）

1. 在"小说编辑器"标签中
2. 点击"批量生成"
3. 设置生成范围（第1-10章）
4. 系统自动逐章生成

## 连贯性系统

### 角色状态跟踪

**功能**：
- 自动记录每个角色的性格、情绪、位置、人际关系
- 追踪角色在各章节中的状态变化
- 检测角色行为的一致性

**查看方法**：
1. 切换到"连贯性分析"标签
2. 选择"👥 角色状态"
3. 从下拉列表选择角色
4. 查看详细信息

### 剧情线管理

**功能**：
- 管理主线、副线、角色线、感情线等
- 追踪伏笔和悬念
- 显示剧情进展时间线

**查看方法**：
1. 切换到"连贯性分析"标签
2. 选择"📖 剧情线"
3. 从下拉列表选择剧情线

### 世界观数据库

**功能**：
- 维护地点、物品、规则等设定
- 检查世界观一致性
- 提供世界观查询

### 连贯性验证

**功能**：
- AI自动检查章节连贯性
- 验证角色性格一致性
- 验证剧情逻辑
- 验证世界观设定

**使用方法**：
1. 切换到"连贯性分析"标签
2. 选择"✅ 连贯性验证"
3. 输入章节号
4. 点击"🔍 验证连贯性"

## 提示词管理

### 查看预设模板

1. 打开"提示词编辑器"标签
2. 选择类别
3. 从模板列表中选择
4. 查看内容

### 自定义提示词

1. 选择模板
2. 在内容框中修改
3. 点击"💾 保存模板"

### 可用变量

- `{title}` - 小说标题
- `{genre}` - 小说类型
- `{chapter_num}` - 章节号
- `{chapter_title}` - 章节标题
- `{chapter_desc}` - 章节描述
- `{context}` - 前文回顾
- `{target_words}` - 目标字数

## 常见问题

### Q1: 生成速度慢怎么办？

**A**: 
- 使用更快的模型（如Groq、Ollama）
- 减少`target_words`目标字数
- 降低`temperature`参数

### Q2: 连贯性评分低怎么办？

**A**:
1. 查看"连贯性验证"报告
2. 检查错误和警告
3. 手动编辑修复问题
4. 重新生成章节

### Q3: 如何修改已生成的章节？

**A**:
1. 在"小说编辑器"中选择章节
2. 在内容框中直接编辑
3. 点击"💾 保存修改"
